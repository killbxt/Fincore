О разработке

Предназначение

Этот проект разработан в рамках курсового проекта для демонстрации и практического применения современных веб-технологий. Цель проекта – создание веб-приложения по основам современного банка. В процессе работы над проектом были использованы и изучены такие технологии, как Angular, Taiga UI, MSSQL, EF Core, и ASP.NET Core API.

Архитектура

Проект построен на основе клиент-серверной архитектуры. Фронтенд разработан с использованием и для создания интуитивно понятного пользовательского интерфейса. Бэкенд реализован на с использованием для взаимодействия с базой данных . API Core обеспечивает интерфейс для обмена данными между клиентом и сервером.

Ключевые особенности

  • Интуитивный интерфейс: Пользовательский интерфейс разработан с использованием Taiga UI, что обеспечивает удобство и простоту использования.
  • Надежная база данных: MSSQL обеспечивает надежное хранение данных и высокую производительность.
  • Современная архитектура: Использование Angular, ASP.NET Core, и EF Core обеспечивает масштабируемость и поддерживаемость приложения.
